[English] Road Geometry Data Standard

1. NDS (Navigation Data Standard)

Paradigm: High-performance binary topology model. Uses Delta-encoding for WGS84 coordinate compression.

2. ASAM OpenDRIVE

Paradigm: Parametric reference line model (s/t coordinate system). Uses splines and polynomials for smooth trajectory generation.

3. Lanelet2

Paradigm: Physical boundary-based hull. Defines lanes via left/right Linestrings, highly semantic for perception.
